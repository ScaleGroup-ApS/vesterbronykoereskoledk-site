# Course Model & Enrollment Date Picker Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Introduce a `Course` model (Offer + start date), let admins manage course dates on the offer edit page, and replace the free-text start date field on the enrollment form with a shadcn day picker that only enables dates where a course exists.

**Architecture:** A new `courses` table holds `(offer_id, start_date, max_students?)`. The `enrollment_requests` table gains a `course_id` FK. `EnrollmentController@show` passes available dates as an Inertia prop; the enroll form submits `course_id` instead of `start_date`. Admins add/remove course dates inline on the offer edit page.

**Tech Stack:** Laravel 12, Pest 4, Inertia v2, React 19, Tailwind CSS v4, shadcn/ui (calendar + popover via `npx shadcn@latest add`), react-day-picker (installed by shadcn), Wayfinder.

```php
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Builder;
use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Relations\BelongsTo;
use Illuminate\Database\Eloquent\Relations\HasMany;

class Course extends Model
{
    /** @use HasFactory<\Database\Factories\CourseFactory> */
    use HasFactory;

    protected $fillable = [
        'offer_id',
        'start_date',
        'max_students',
    ];

    protected function casts(): array
    {
        return [
            'start_date' => 'date',
            'max_students' => 'integer',
        ];
    }

    public function offer(): BelongsTo
    {
        return $this->belongsTo(Offer::class);
    }

    public function enrollmentRequests(): HasMany
    {
        return $this->hasMany(EnrollmentRequest::class);
    }

    /** @param Builder<Course> $query */
    public function scopeUpcoming(Builder $query): void
    {
        $query->where('start_date', '>=', now()->toDateString());
    }
}
```

/**
 * @param  array{name: string, email: string, password: string, phone?: string|null, cpr?: string|null, course_id: int, payment_method: string}  $data
 * @return array{0: Student, 1: EnrollmentRequest}

public function handle(array $data, Offer $offer): array
{
    return DB::transaction(function () use ($data, $offer) {
        $course = \App\Models\Course::findOrFail($data['course_id']);

        $user = User::create([
            'name' => $data['name'],
            'email' => $data['email'],
            'password' => Hash::make($data['password']),
            'role' => UserRole::Student,
        ]);

        $student = Student::create([
            'user_id' => $user->id,
            'phone' => $data['phone'] ?? null,
            'cpr' => $data['cpr'] ?? null,
            'status' => StudentStatus::Active,
            'start_date' => $course->start_date->toDateString(),
        ]);

        StudentEnrolled::fire(
            student_id: $student->id,
            student_name: $user->name,
            start_date: $student->start_date->toDateString(),
        );

        $paymentMethod = EnrollmentPaymentMethod::from($data['payment_method']);

        $status = $paymentMethod === EnrollmentPaymentMethod::Stripe
            ? EnrollmentStatus::PendingPayment
            : EnrollmentStatus::PendingApproval;

        $enrollmentRequest = EnrollmentRequest::create([
            'student_id' => $student->id,
            'offer_id' => $offer->id,
            'course_id' => $course->id,
            'payment_method' => $paymentMethod,
            'status' => $status,
        ]);

        EnrollmentRequested::fire(
            enrollment_request_id: $enrollmentRequest->id,
            student_id: $student->id,
            offer_id: $offer->id,
            payment_method: $paymentMethod->value,
        );

        return [$student, $enrollmentRequest];
    });
}
